# Project Overview: MindMap Manifesto

**Version**: 2026-01-26  
**Last Updated**: 2026-01-26

A collection of interactive tools and high-conversion landing pages for the MindMap Manifesto ecosystem.

## Core Components

### 1. Interactive Intake Forms
*   **Purpose**: Multi-page hypnotherapy intake sequence.
*   **Features**:
    *   3-page narrative journey.
    *   Automatic local saving of progress.
    *   Integration with EmailJS for serverless form submissions.
*   **Files**: `hypnotherapy_intake_page1.html` to `page3.html`.

### 2. GEO Agency Landing Pages
*   **Purpose**: Narrative-driven landing pages for agency services.
*   **Evolution**: Consolidated multiple versions into a unified 3-page flow.
*   **Files**: `geo-agencythree-page1.html` to `page3.html`.

### 3. Unified Therapeutic Script
*   **Purpose**: A React-powered interactive tool for therapeutic sessions.
*   **Features**:
    *   Diagnostic "Cheat Sheets" for symptom analysis.
    *   Structured regression and transformation protocols.
    *   Micro-animations and modern UI for premium feel.
*   **File**: `unified.html`

### 4. Technical Configuration
*   **Email Handling**: Configured via EmailJS (Instructions in `EMAIL_SETUP_INSTRUCTIONS.md`).
*   **Version Control**: Synchronized with GitHub repository (`rootvisionx/mindmapmanifesto`).
*   **Styling**: Built with modern CSS and Tailwind CSS.
