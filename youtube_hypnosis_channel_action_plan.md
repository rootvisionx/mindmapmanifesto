## PHASE 3: VIDEO PRODUCTION SYSTEM (Week 3-4)

### ‚úÖ Production Workflow

**Action 3.1: Equipment Setup**
- [ ] Camera: Smartphone (minimum) or DSLR/mirrorless

- [ ] Backdrop: Clean, minimal background OR green screen

**Action 3.2: Recording Setup for Different Video Types**

**Type A: Talking Head (Educational)**
- [ ] Frame: Medium close-up (head and shoulders)
- [ ] Lighting: Soft, even, three-point if possible
- [ ] Audio: Crystal clear (test and re-test)
- [ ] Background: Minimal, professional
- [ ] Duration: 8-15 minutes

**Type B: Guided Hypnosis**
- [ ] Frame: Close-up of face OR just background visuals
- [ ] Lighting: Dim, ambient, warm tones
- [ ] Audio: PARAMOUNT - soothing, clear, ASMR-quality
- [ ] Background: Calming visuals (moving clouds, water, sacred geometry)
- [ ] Duration: 10-30 minutes
- [ ] Music: Binaural beats, 432Hz, theta waves

**Type C: YouTube Shorts**
- [ ] Frame: Vertical 9:16 format
- [ ] Hook: First 1 second must grab attention
- [ ] Text: Large, bold captions
- [ ] Duration: 15-60 seconds
- [ ] Energy: High intensity, fast cuts

- [ ] Test with headphones before publishing

## TROUBLESHOOTING GUIDE

### Problem: Videos aren't getting views
**Solutions:**
- [ ] Improve thumbnail (make it more eye-catching)
- [ ] Rewrite title (use proven formula)
- [ ] Check if keyword has search volume
- [ ] Make stronger hook in first 5 seconds
- [ ] Promote in relevant communities

### Problem: Low watch time
**Solutions:**
- [ ] Cut intro shorter (under 10 seconds)
- [ ] Remove fluff and filler
- [ ] Add pattern interrupts (every 30 seconds)
- [ ] Improve pacing (speak faster or cut tighter)
- [ ] Add visual interest (B-roll, graphics)

### Problem: Not growing subscribers
**Solutions:**
- [ ] Ask for subscription in every video
- [ ] Make the ask specific: "If you want more hypnosis techniques, subscribe"
- [ ] Create content series (reason to sub = more episodes)
- [ ] Make consistently valuable content
- [ ] Engage more with community

### Problem: Low engagement
**Solutions:**
- [ ] Ask specific questions
- [ ] Create polls in community tab
- [ ] Respond to every comment
- [ ] Ask viewers to comment specific word
- [ ] Create "comment to unlock" strategy
